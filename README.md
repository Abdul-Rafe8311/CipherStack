# 🔐 CipherStack

A visual node-based cascade encryption builder that allows users to construct, execute, and visualize multi-layer encryption pipelines in an interactive way.

---

## 🚀 Overview

CipherStack is a modern frontend application that demonstrates the concept of **cascade encryption**, where data is passed through multiple cipher algorithms sequentially. Each transformation adds a layer of complexity, making the final output more secure.

Users can visually build a pipeline of encryption nodes, configure each algorithm, and observe how data evolves step by step.

---

## ✨ Features

* 🧩 Node-based encryption pipeline
* 🔁 Supports both **Encryption & Decryption**
* ⚙️ Configurable ciphers (Caesar, XOR, Vigenere)
* 🔍 Intermediate outputs for every node
* ⚡ Real-time pipeline execution
* 🌗 Dark / Light mode support
* 🎬 Animated data flow (visual pipeline execution)
* 📋 Copy output functionality

---

## 🛠️ Tech Stack

* **React**
* **Tailwind CSS**
* **Framer Motion**
* **Vite**

---

## 🧠 How It Works

1. Enter plaintext in the input panel
2. Add cipher nodes (Caesar, XOR, Vigenere)
3. Configure each node (shift, key, keyword)
4. Run the pipeline
5. View final encrypted output
6. Switch to decrypt mode to recover original text
